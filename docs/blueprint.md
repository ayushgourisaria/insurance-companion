# **App Name**: Clarity Cover

## Core Features:

- Coverage Overview: Display a clear, at-a-glance summary of the user's insurance plan, deductible progress, and out-of-pocket expenses.
- AI Assistant: A Gemini-powered chatbot that helps users understand their coverage, prepare for visits, and understand claim decisions. Acts as a tool to retrieve specific, easily digestible policy details.
- Claims List: Show a list of insurance claims with key details like provider, service, date, and status.
- Claim Details: Detailed view of individual claims, including service breakdown and plain-English explanation of the decision.
- Smart Navigation: Easy access to key features with a bottom navigation bar: Home, Claims, AI Assistant, and Profile.
- Mode-Specific Gemini Prompts: Send mode-specific prompts to Gemini API based on the user's selected use case (Coverage & Benefits, Visit Preparation, Claims Discussion).
- Profile Management: Display the user's profile information (member name, plan name, member ID) and provide a Contact Support button.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke trust and calmness.
- Background color: Very light blue (#E3F2FD), almost white, to create a clean, reassuring feel.
- Accent color: Pale green (#A5D6A7), a close analog to blue, to suggest reassurance and well-being.
- Font pairing: 'Poppins' (sans-serif) for headlines and shorter content, 'PT Sans' (sans-serif) for body text to create a readable and modern feel.
- Use clear, simple icons that are easily understandable to represent insurance concepts and actions.
- Employ a card-based layout with rounded corners for a modern and friendly user interface.
- Subtle, gentle animations on state changes and loading to enhance user experience.
